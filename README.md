# 📊 Dashboard PCM – Controle de Ordens e Apontamento de Horas

Sistema desenvolvido para apoio ao Planejamento e Controle de Manutenção (PCM), com foco em organização, rastreabilidade e controle de horas da equipe.

🔗 Acesse o sistema:
https://fernandors99.github.io/DashPcmSite/

---

## 🎯 Objetivo

Centralizar e facilitar o controle de ordens de manutenção, permitindo:

- Apontamento de horas da equipe
- Visualização de ordens (preventivas e corretivas)
- Apoio à tomada de decisão com base em dados
- Redução de falhas manuais no processo

---

## ⚙️ Funcionalidades

- 📌 Dashboard de ordens de manutenção
- ⏱️ Controle e apontamento de horas trabalhadas
- 📊 Indicadores visuais (preventiva x corretiva)
- 👥 Controle de usuários (admin e operadores)
- 📱 Interface responsiva (desktop e mobile)

---

## 🧠 Conceito aplicado

O sistema foi pensado com base em práticas de:

- PCM (Planejamento e Controle de Manutenção)
- TPM (Manutenção Produtiva Total)
- Melhoria contínua (redução de variabilidade e erros humanos)

---

## 🚀 Tecnologias utilizadas

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Banco de dados: SQLite
- Versionamento: Git + GitHub
- Deploy: GitHub Pages

---

## 📈 Benefícios operacionais

- Maior controle sobre horas apontadas
- Redução de retrabalho e inconsistências
- Melhor visibilidade das ordens de manutenção
- Apoio na gestão de equipe e produtividade
